# The movie search

## Development
```shellscript
npm install && npm start
# or
yarn install && yarn start
```

## Webserver static serving
```shellscript
npm run server
# or
yarn server
```

## Build
```shellscript
npm run build
# or
yarn build
```

## Push to heroku
```shellscript
git push heroku ${branch}
```
